# EKINE
**Inspirando Calidad y Cultura**

Proyecto web desarrollado para mostrar servicios de identidad territorial y exportación premium.

## Características:
- ✅ **Acordeones con colores distintivos**: Dorado para consulta, rojo para trials
- ✅ **Header optimizado**: Logo a la izquierda, título centrado
- ✅ **Navegación suave**: Scroll perfecto que muestra subtítulos completos
- ✅ **Responsive design**: Adaptado para móviles y desktop
- ✅ **Calculadora ROI integrada**
- ✅ **Formularios embebidos de Google Forms**

## Cómo usar:
1. Abrir `index.html` en cualquier navegador
2. Los acordeones dorado (consulta) y rojo (trials) destacan las acciones principales
3. La navegación del header lleva a cada sección mostrando títulos completos

## Mejoras implementadas:
- Header compacto (90px) vs anterior (140px)
- JavaScript personalizado para scroll suave
- Colores distintivos en acordeones para mejor UX
- Layout optimizado logo + título
